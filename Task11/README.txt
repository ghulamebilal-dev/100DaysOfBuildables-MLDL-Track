# Model Comparison Task

This project trains and compares three models on the Iris dataset:
- Logistic Regression (Baseline)
- Decision Tree (Alternative)
- K-Nearest Neighbors (Alternative)

## Files Generated:
- model_comparison.csv: Accuracy comparison table
- predictions.csv: Actual vs predicted classes
- decision_tree_confusion_matrix.png: Visual confusion matrix

## How to Run:
1. Install dependencies:
   pip install scikit-learn pandas matplotlib seaborn

2. Run the script:
   python model_comparison.py

3. Output files will appear in the same folder. Zip them and submit.
